## Github 아이디 연동

인텔리제이에서 지원하는 다양한 기능들을 사용하기 위해서는 Github 연동이 반드시 필요 합니다.

![](https://i.imgur.com/wdn4bEW.png)

1. 설정에서 github 텝 인동
2. 해당 Auth Type 선택 후 Test 버튼 클릭
3. Success 출력 확인후 OK 버튼 클릭
